دوره‌های آموزشی

پایتون

پایتون یک زبان برنامه‌نویسی سطح بالا و تفسیر شده است که به خاطر خوانایی و تطبیق‌پذیری آن شناخته می‌شود. پایتون در زمینه‌های مختلفی مانند وب، داده، یادگیری ماشین، الگوریتم و در بسیاری از زمینه ها دیگر کابرد دارد.

شروع یادگیری

HTML

HTML (زبان نشانه‌گذاری) زبان استاندارد برای ایجاد صفحات وب است. این زبان ساختار محتوای وب را تعریف می‌کند. با یادگیری HTML می‌توانید ساختار اولیه وبسایت‌ها را ایجاد کنید.

CSS

CSS (شیوه‌نامه آبشاری) برای استایل‌دهی و چیدمان صفحات وب استفاده می‌شود. با CSS می‌توانید ظاهر زیبا و واکنش‌گرا برای سایت خود طراحی کنید.

JavaScript

جاوااسکریپت زبان برنامه‌نویسی وب است که به صفحات وب تعامل و پویایی می‌بخشد. این زبان برای توسعه فرانت‌اند و بک‌اند استفاده می‌شود و با یادگیری جاوااسکریپت می‌توانید وبسایت‌های تعاملی ایجاد کنید.

About Us

Herat Code School is an educational platform launched with the aim of empowering Afghan girls. It strives to create a path for Afghans to enter the world of technology and programming. The training courses are designed in a way that makes learning possible from basic to advanced levels, preparing you to develop your personal skills.
